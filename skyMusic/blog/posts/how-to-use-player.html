Guide

The player is meant to help you record a song by hand or practice an existing song with the two practice tools.
You can also import sheets made in the app / record audio of the keyboard and play freely, you also have a metronome to help you with the tempo.
If you want a simpler keyboard you can use the Zen keyboard which has all the instruments and pitch settings, but bigger keyboard and simpler interface.
In the settings you can change the instrument, pitch, reverb, volume, keyboard size, etc... If you have a MIDI keyboard, you can connect it to your device, follow the MIDI device

How to use the player

The images below are from the genshin version of the app, but the functionality is the same

  1. Pressing this button will start an audio recording, everything you play from that moment will be recorded, then downloaded as an audio file
  2. When you start practicing or playing a song, this bar on the left will tell you the progress of the song. You can move the start and end markers to set where to make the song/practice start and end
  3. This selects the playback speed of the song, useful when you are trying to play along with the approaching circles learning tool
  4. Together with point (2), this will restart the song from the selected start point
  5. When in practice mode, this is a note that you should press now. When this note appears, it will fade in red, when the fade in finishes, it will indicate the moment when you should press the note to stay in tempo
  6. This indicator tells you that this note will be pressed after you finished pressing all the current red notes.
  7. This opens the song library, it's a small library made years ago by the sky music community
  8. View your songs
  9. Opens the menu of the app where you can go to all the other pages
  10. Visual sheet of the song

When you open the "song menu", you will be shown this window. Your songs will be split between recorded and composed songs. The recorded songs are the ones which were recorded by hand, the composed songs are the ones which were composed using the composer. Here you can select the song to play/practice. Or download, change, rename and move between your folders.

  1. This is a song that you have saved in the app, by pressing the song name, you will start playing the song.
  2. Pressing this button will start the song practice, the notes will become red, showing which notes you have to press in that moment. This should be used to learn a song, you can select sections to replay by using the notes selection bar on the right.
  3. Pressing this button will start the second kind of practice, the approaching circles, circles will appear coming towards the note, once they reach it, you should press the note. This is meant to be used once you are more comfortable with the song, it is harder to play compared to the practice mode, but will test your muscle memory.
  4. Opens the menu for that song, showing options to rename, move to a folder, edit, download and delete the song.
  5. By clicking this you can select in which folder to move this song.
  6. Those two are the download buttons, the first one is the suggested one to share it with other people, or to backup your songs. The second one downloads the song as a MIDI song, this will convert it to MIDI, but it will cause some information to be lost, as MIDI cannot store some information that is needed by the app, use it only if you need to import it in another app which is not this one.

Player settings

  1. Instrument: This is the instrument of the keyboard, you can change the volume
  2. Pitch: This is the pitch of the instrument
  3. BPM: The BPM which will be saved when you record the song, also used for the metronome
  4. Auto sync the song's instrument and pitch: When selecting a song to play, change the current instrument and pitch to match the song
  5. Metronome beats: Set how many beats to use in the metronome, every nth beats there will be a different sound played
  6. Metronome volume: The volume of the metronome
  7. Reverb (cave mode): Sets if the instrument should use reverb or not.
  8. Note name type: Sets the kind of text to show inside of the note
  9. Keyboard size: Scales the size of the keyboard up/down
  10. Keyboard vertical position: Pushes the keyboard up or down
  11. Approach rate: When playing the "approaching circles" mode, this is how many ms the note will take from when it appears, to when it should be pressed.
  12. Note animation: Show an animation when you press the note
  13. Show visual sheet: Show the visual sheet above the keyboard. This helps to visualize the notes that are about to be played. Useful both in practice and playback of a song.

Player shortcuts

The player has some shortcuts you can use, if you want to change them, go to the keybinds page

Space
Toggle keyboard recording
ShiftLeft+KeyS
Stop playing / recording / practicing
ShiftLeft+KeyR
Restart the song or practice
ShiftLeft+KeyM
Toggle the menu
Escape
Close the menu
