Song Settings

InstrumentThe main (first) instrument of the player, will also be saved in the song you record
PitchThe pitch of the player, will also be saved in the song you record
BpmBeats per minute, used by the metronome and will be used when converting the song with the compsoer

Player Settings

Auto sync the song's instruments and pitchWhenever you load a song, the instruments and pitch of that song will be loaded too
Metronome beatsAfter how many times a stronger beat is played
Metronome volumeThe volume of the metronome
Reverb (cave mode)Makes it sound like you are in a cave
Note name typeThe type of text which will be written on the note
Keyboard sizeScales the keyboard size
Keyboard vertical positionThe vertical position of the keyboard
Approach Rate (AR)The time between when the notes appear and when they reach the end (in ms)
Note animationToggle the animation of the notes (will reduce performance)
Show visual sheetShow the sheet above the keyboard (might reduce performance)
Storage is not persistedThe browser didn't allow to store data persistently, it might happen that you will loose data when cache is automatically cleared. To get persistent storage, add the app to the home screen. If that still doesn't work, make sure you do a backup often

Main page
  1. Record the keyboard as an audio file
  2. The selection where you want the song to start/end
  3. Playback speed
  4. Restart the song from the selected starting point
  5. Practice tool note, you should press this note
  6. Practice tool note, this will need to be clicked after all the red ones
  7. Public Song library
  8. Your songs
  9. Open the Page selection menu
Main page shortcuts
The main page has shortcuts for PC users:
Shift + R
reload song
Shift + C
Start/Stop recording when there are no songs playing
Shift + M
Open/Close the menu
Esc
 Close the menu
Shift + S
stop song
Composer
  1. Go to the next / previous breakpoint
  2. Timeline of the breakpoints
  3. Open the tools
  4. Add 16 columns to the end
  5. Remove the current selected column
  6. Add column after the current one
  7. Layer (instrument) selection
  8. Change column's duration
